FAQ

Whom is this for?

This workshop is for everyone who wants to learn ClojureScript. But mainly it targets JavaScript developers.

What I'll learn?

We will go through language basics starting with the syntax and all the way to building a simple web app. You'll learn about language constructs, best practices, how to setup a project, use REPL and finally how to build a web app with ClojureScript.

Is it free?

Completely! But we have a limited amount of seats and also a set of requirements to make sure that all participants can learn efficiently.

How long is it going to be?

6-8 hours. We are starting at 10:00 AM, so don't be late.

Do I need to bring my laptop?

Yes. We are going to write code. Also you'll be given test assignments.

What about lunch?

The host company Cogniance was very kind to provide us free pizza.

Requirements

It's nice to be familiar with the following:

  • React.js library (learn)
  • State management solutions like Flux (learn)
  • A concept of immutability (learn)
  • Basics of functional programming (learn)
